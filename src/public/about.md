## About
### 개요
- 여러 기능을 짬뽕한 사이트를 만들고자 함
- 디자인은 최소화하고 백 서버에서 API 호출하여 서비스

### 기능
- Home: 메인 페이지, 현재는 깃허브 블로그의 최신 목록을 가져옴
- Labs: gpt 호출, 마크다운 변환 등 여러 기능들을 다뤄볼 핵심 페이지
- Trend: 구글 트렌드 인기검색어 등 사람들의 관심 키워드를 수집해서 통계하는 페이지
